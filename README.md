# Restaurant & Consumer Data Analysis

## Project Overview

This project analyzes restaurant and consumer data to understand customer behavior, restaurant ratings, and cuisine preferences. The analysis helps identify trends that can assist restaurants in improving services and making data-driven decisions. The project uses SQL for data analysis, Excel for data preparation, and Power BI for visualization.

## Objectives

* Analyze restaurant ratings given by consumers
* Identify popular cuisines and restaurant types
* Understand consumer preferences and behavior
* Evaluate restaurant performance based on ratings
* Generate insights using data visualization

## Dataset Description

The project uses multiple datasets related to restaurants and consumers.

| Dataset                  | Description                                           |
| ------------------------ | ----------------------------------------------------- |
| ratings.csv              | Contains ratings given by customers to restaurants    |
| restaurants.csv          | Restaurant information including location and details |
| restaurant_cuisines.csv  | Types of cuisines served by restaurants               |
| consumers.csv            | Customer demographic information                      |
| consumer_preferences.csv | Food preferences of consumers                         |

## Tools & Technologies

* SQL – Data querying and analysis
* Excel – Data cleaning and preparation
* Power BI – Data visualization and dashboard creation

## Key Analysis Performed

* Average rating analysis for restaurants
* Identification of top rated restaurants
* Cuisine popularity analysis
* Consumer preference trends
* Restaurant performance comparison

## Data Visualization

A Power BI dashboard was created to visualize insights such as:

* Restaurant rating distribution
* Cuisine popularity
* Consumer preference patterns
* Restaurant performance comparison
